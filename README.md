# LLM From Scratch

A decoder-only Transformer implementation built in PyTorch while following Andrej Karpathy's educational approach.

## Features

- Character-level tokenizer
- Token embeddings
- Positional embeddings
- Self-attention
- Multi-head attention
- Feed-forward network
- LayerNorm
- Residual connections
- Decoder-only Transformer
- Autoregressive text generation
